# Changelog

## [1.0.0] — 2026-04-08

### Added
- Chat Participant `@fleet` с командами `/run`, `/plan`, `/status`, `/abort`
- Декомпозиция задачи через VS Code LM API (Copilot)
- Создание Issues в целевом репозитории
- Назначение copilot-swe-agent[bot] на Issues
- Мониторинг PR с polling
- Sidebar Dashboard с реальным временем прогресса
- Status Bar Item с иконками состояния
- Output Channel для подробных логов
- DAG зависимости между подзадачами
- Пресеты: solo, squad, platoon, fleet
- `--agents N` параметр в чате
- Настройка через VS Code Settings
- Поддержка тёмной и светлой тем VS Code
- Persistence истории сессий через globalState
